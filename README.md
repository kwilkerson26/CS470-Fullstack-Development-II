# CS470-Fullstack-Development-II

This course helped me connect a lot of what I’ve learned in software development into a real cloud-based workflow. One of the biggest takeaways for me was understanding how full-stack applications can be designed and deployed using cloud services instead of relying on traditional server setups. Working through the process of building, testing, and documenting a cloud-native application helped me see how development changes when you focus on scalability, cost, and maintenance.
